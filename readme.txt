This plugin notifies the comment auther by email on approval of his/her Comment.

== Description ==

This is a WordPress Plugin for inform a comment author on the approval of author comment on WordPress site.
This Plugin have a admin panel setting page where you can add CC and BCC for mail.

This plugin will send a HTML email content so you can send content design as you want.

This Plugin have a HTML editor in its setting page section to design mail body.

You can insert code for site URL, Comment page URL And Site name as dynamic mail content . You can do it by inserting [site_url], [comment_url] and [site_name].
Key Featurs

    HTML editor
    HTML email
    Dynamic URLs in mail
    Option to add CC and BCC
    Email Content Editor


== Installation ==

1. Upload `email-notification-on-comment-approval` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now it will work you can change default settings by wp-admin/admin.php?page=enocp_settings.
