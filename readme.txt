This plugin notifies the comment auther by email on approval of his/her Comment.

== Description ==

This Plugin Notifies a comment author by email on approval of his/her comment.



== Installation ==

1. Upload `email-notification-on-comment-approval.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now it will work you can change default settings by wp-admin/admin.php?page=enocp_settings.
